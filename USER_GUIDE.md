## 🏠 欢迎使用 Pigeon Planner

Pigeon Planner 是一个多平台赛鸽管理应用程序，为赛鸽爱好者提供完整的鸽群管理、血统追踪、繁殖管理和比赛记录功能。

## 📋 系统要求

### 支持的平台
- **macOS**: 10.15+ (Catalina 或更高版本)
- **Windows**: Windows 10 或更高版本
- **Linux**: Ubuntu 18.04+ 或同等发行版

### 硬件要求
- **内存**: 最低 4GB RAM
- **存储**: 最低 100MB 可用空间
- **处理器**: 64位处理器

## 🚀 安装指南

### macOS 安装
1. 下载 `Pigeon Planner_1.0.0_aarch64.dmg` 文件
2. 双击 DMG 文件打开安装器
3. 将 Pigeon Planner 拖拽到应用程序文件夹
4. 从启动台或应用程序文件夹启动应用

### Windows 安装
1. 下载 `PigeonPlanner_1.0.0_x64_setup.exe` 文件
2. 双击安装程序
3. 按照安装向导完成安装
4. 从开始菜单或桌面快捷方式启动应用

### Linux 安装
1. 下载 `PigeonPlanner_1.0.0_amd64.AppImage` 文件
2. 给文件添加执行权限：`chmod +x PigeonPlanner_1.0.0_amd64.AppImage`
3. 双击文件启动应用

## 🎯 核心功能

### 1. 鸽子管理
**位置**: 导航栏 → 鸽子管理

#### 添加新鸽子
1. 点击"添加鸽子"按钮
2. 填写基本信息：
   - 环号（必填）
   - 名字
   - 性别
   - 品种
   - 羽色
   - 出生日期
3. 点击"保存"

#### 查看鸽子信息
- 在鸽子列表中点击任意鸽子查看详细信息
- 支持按名字、环号等条件搜索

#### 编辑和删除
- 在鸽子详情页面点击"编辑"按钮修改信息
- 点击"删除"按钮移除鸽子记录

### 2. 血统管理
**位置**: 导航栏 → 血统管理

#### 查看血统书
- 选择任意鸽子查看其家族血统
- 支持多代血统追溯
- 显示父子关系和遗传系数

#### 血统分析
- 自动计算亲缘关系
- 识别共同祖先
- 提供配对建议

### 3. 繁殖管理
**位置**: 导航栏 → 繁殖管理

#### 配对管理
1. 点击"新建配对"
2. 选择公鸽和母鸽
3. 设置配对日期和巢箱
4. 添加繁殖备注

#### 繁殖记录
- 记录产蛋日期、孵化日期
- 追踪雏鸟成长情况
- 管理巢箱分配

### 4. 比赛管理
**位置**: 导航栏 → 比赛成绩

#### 创建比赛
1. 点击"新建比赛"
2. 填写比赛信息：
   - 比赛名称
   - 比赛日期
   - 比赛距离（公里）
   - 比赛类型
   - 天气条件
3. 点击"保存"

#### 报名参赛
- 从鸽子列表选择参赛者
- 批量报名功能
- 自动生成参赛编号

#### 录入成绩
1. 进入比赛详情页面
2. 点击"录入成绩"
3. 输入归巢时间和排名
4. 系统自动计算飞行速度

## 💾 数据管理

### 数据备份
Pigeon Planner 自动将数据保存在本地：
- **macOS**: `~/Library/Application Support/Pigeon Planner/`
- **Windows**: `%APPDATA%/Pigeon Planner/`
- **Linux**: `~/.config/Pigeon Planner/`

### 数据导出
**经典版本**支持导出功能：
- 鸽子列表 (CSV 格式)
- 比赛成绩 (PDF 报告)
- 血统图表 (PNG 格式)

**现代化版本**：数据导出功能待实现中

### 数据导入
支持从其他赛鸽软件导入数据（通过 CSV 文件）

## 🆘 故障排除

### 常见问题

#### 1. 应用无法启动
- 检查系统是否满足最低要求
- 重新安装应用程序
- 联系技术支持

#### 2. 数据丢失
- 检查数据备份文件
- 使用数据导入功能恢复
- 定期备份重要数据

#### 3. 性能问题
- 关闭不必要的后台程序
- 清理历史数据
- 增加系统内存

### 错误代码
- **E001**: 数据库连接失败
- **E002**: 文件权限错误
- **E003**: 数据验证失败
