# 🕊️ Pigeon Planner

现代化的赛鸽管理应用程序，为赛鸽爱好者提供完整的鸽群管理、血统追踪、繁殖管理和比赛记录功能。

## ✨ 特性

### 🏆 全新比赛管理模块
- ✨ 完整的比赛创建和管理功能
- 📊 自动飞行速度计算
- 🏅 比赛成绩排名系统
- 📋 批量参赛者报名
- 📈 比赛统计分析

### 🐦 智能鸽子管理
- 📝 详细的信息记录系统
- 🏷️ 多维度标签分类
- 🔍 强大的搜索和过滤功能
- 📸 照片管理支持

### 👨‍👩‍👧‍👦 专业血统管理
- 🌳 多代血统树可视化
- 🔗 亲缘关系智能计算
- 💡 AI辅助配对建议
- 📊 血统分析报告

### 🥚 科学繁殖管理
- ❤️ 配对管理系统
- 🏠 巢箱分配和状态监控
- 📈 繁殖成功率统计分析
- 📋 详细繁殖记录追踪

## 🚀 快速开始

### 下载安装

#### macOS
```bash
# 下载并挂载 DMG 文件
open Pigeon Planner_1.0.0_aarch64.dmg
```

#### Windows
```bash
# 运行安装程序
PigeonPlanner_1.0.0_x64_setup.exe
```

#### Linux
```bash
# 下载并运行 AppImage
chmod +x PigeonPlanner_1.0.0_amd64.AppImage
./PigeonPlanner_1.0.0_amd64.AppImage
```

### 5分钟快速上手
1. 🚀 启动应用
2. 🐦 添加第一只鸽子
3. 👨‍👩‍👧‍👦 建立血统关系
4. 🥚 配对繁殖
5. 🏆 创建比赛

详细说明请查看 [快速开始指南](QUICK_START.md)

## 🛠️ 开发

本项目采用现代化开发工作流：

```bash
# 克隆仓库
git clone https://github.com/pigeonplanner/pigeon-planner.git
cd pigeon-planner

# 进入现代化版本目录
cd pigeon-planner-rs

# 安装依赖
npm install
cargo install tauri-cli

# 启动开发服务器
npm run tauri:dev

# 构建生产版本
npm run tauri:build
```
