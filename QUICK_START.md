# Pigeon Planner 快速开始指南

## 🚀 5分钟快速上手

### 1. 启动应用
双击桌面图标或从应用程序菜单启动 Pigeon Planner

### 2. 添加第一只鸽子
1. 点击导航栏"鸽子管理"
2. 点击右上角"+ 添加鸽子"
3. 输入环号和名字
4. 选择性别、品种等基本信息
5. 点击"保存"

### 3. 建立血统关系
1. 在鸽子详情页面点击"编辑血统"
2. 选择父鸽（公鸽）
3. 选择母鸽（母鸽）
4. 系统自动生成血统树

### 4. 配对繁殖
1. 进入"繁殖管理"
2. 点击"新建配对"
3. 选择要配对的公鸽和母鸽
4. 设置配对日期
5. 分配巢箱

### 5. 创建比赛
1. 导航到"比赛成绩"
2. 点击"新建比赛"
3. 输入比赛名称和距离
4. 从列表中选择参赛鸽子
5. 点击"开始比赛"

## 💡 核心功能一览

| 功能 | 位置 | 用途 |
|------|------|------|
| 🐦 鸽子管理 | 导航栏 | 管理鸽子基本信息 |
| 👨‍👩‍👧‍👦 血统管理 | 导航栏 | 查看和编辑家族关系 |
| 🥚 繁殖管理 | 导航栏 | 配对和繁殖记录 |
| 🏆 比赛成绩 | 导航栏 | 比赛组织和成绩记录 |
| 📊 统计分析 | 菜单栏 | 查看数据和报告 |

## 🔥 常用操作

### 快速搜索
- 在任意列表页面的搜索框输入关键词
- 支持按名字、环号、品种搜索

### 批量操作
- 按住 Ctrl/Cmd 键选择多个项目
- 支持批量编辑、删除、导出

### 数据备份
- 文件菜单 → 导出数据
- 选择导出格式（CSV/PDF）
- 选择保存位置
