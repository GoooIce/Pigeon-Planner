# Pigeon Planner 变更日志

## 现代化版本 (Rust + Tauri + React)

### v1.0.0 (2025-11-29) - 🎉 首次正式发布

#### ✨ 全新比赛管理模块
- 完整的比赛创建和管理功能
- 自动飞行速度计算
- 比赛成绩排名系统
- 批量参赛者报名功能
- 比赛统计分析

#### 🐦 完善的鸽子管理系统
- 详细的信息记录
- 多维度标签分类
- 强大的搜索功能
- 照片管理支持

#### 👨‍👩‍👧‍👦 专业血统管理
- 多代血统树展示
- 亲缘关系计算
- 智能配对建议
- 血统分析报告

#### 🥚 繁殖管理工具
- 配对管理系统
- 巢箱分配管理
- 繁殖成功率统计
- 繁殖记录追踪

#### 🎨 现代化技术栈
- **Rust + Tauri** - 高性能跨平台桌面应用
- **React + TypeScript** - 现代化用户界面
- **SQLite + SQLx** - 类型安全的数据库操作
- **shadcn/ui** - 优美的用户界面组件

#### 🔧 数据安全
- 本地数据存储，隐私无忧
- 自动数据备份
- 数据导出功能 (CSV/PDF)
- 数据迁移工具

#### 🌐 跨平台支持
- macOS 10.15+ (原生应用)
- Windows 10+ (原生应用)
- Linux Ubuntu 18.04+ (原生应用)

- ⚠️ 大文件导入时可能需要等待较长时间
